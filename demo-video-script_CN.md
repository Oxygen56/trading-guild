## 第 1 段：钩子（0:00–0:20）

**画面**：左侧人类交易员盯 6 块屏幕（黑白），右侧三个 Agent 头像（Athena 紫、Guardian 金、Mercury 青）脉动发光。文字叠层逐词出现。

**解说**：

> DeFi 交易，你需要同时当策略师、风控师、交易员——7×24 小时，一个人扛所有角色。
>
> 但如果 AI Agent 能搞定一切呢？彼此协作、相互支付、自主交易，每个决策链上可查。
>
> 这就是 Casper 自主交易公会。

**转场**：Logo 放大切入仪表盘。

---

## 第 2 段：面板概览（0:20–0:50）

**画面**：仪表盘深色主题，鼠标依次指向金库卡片 → Agent 卡片 → 动态时间线。约 0:35 叠层架构图。

**解说**：

> 金库管理 50,000 CSPR，三个 Agent 在线。
>
> Athena 扫描市场、生成策略。Guardian 审核风险、定价收费。Mercury 在链上执行交易。
>
> 它们不是聊天机器人——各自有钱包、有定价、有信誉。来看实际运作。

---

## 第 3 段：成功路径（0:50–1:50）

**画面**：终端输入 → Agent 气泡回复 → 三栏展示处理过程 → 底部交易流程图 → Explorer 链接弹出。

**解说**：

> 我在终端输入：「找出 CSPR.trade 上滑点 5% 以内的最优收益。」
>
> Athena 扫描流动性池，找到 CSPR-USDT：TVL 210 万，年化 35.2%。生成 swap 策略：5,000 CSPR 换 USDT。
>
> 但资金动用前必须过风控。Athena 通过 x402 向 Guardian 支付 0.005 CSPR 审核费——链上结算，没有 API Key。
>
> Guardian 做 6 维检查：代币安全、流动性深度、滑点模拟、池集中度、历史波动、近期活跃度。评分 8/100——低风险，通过。
>
> Athena 向 Mercury 支付 0.01 CSPR 执行费。Mercury 调 CSPR.trade DEX，提交 swap，返回 tx hash。Casper Testnet 已确认。
>
> 全程除了最初一行指令，无需人类介入。

**转场**：聚焦支付账本 → 叠层「如果策略太危险呢？」

---

## 第 4 段：风险拒绝（1:50–2:30）

**画面**：新终端输入 → Guardian 面板红色指标 → 评分 51/100 → REJECTED 状态 → 金库余额不变（绿色脉冲）。

**解说**：

> 不是每笔策略都能过。Athena 分析一个新代币——Guardian 发现卖出税 90%，流动性池刚创建 2 小时，部署者通过混币器注资。评分 51，超过拒绝线。
>
> 策略被拦截。审核费 0.005 CSPR 照付——Guardian 尽责了——但金库零风险。
>
> 核心设计：Agent 有切身利益。Guardian 帮公会避坑赚得更多，Mercury 按成功执行收费。经济激励与公会收益完全对齐。

**转场**：点击侧边栏「数据分析」。

---

## 第 5 段：数据透明（2:30–3:10）

**画面**：盈亏图动画 → Agent 绩效表（可排序）→ 利润分配轮次。

**解说**：

> 每个决策、每笔支付、每次交易——数据分析页上完全透明。
>
> 盈亏图追踪公会表现。Agent 绩效表展示每个 Agent 的提案数、通过率、创造利润——这是信誉系统，表现好的 Agent 获得更多资金。
>
> 利润分配算法驱动：公会平仓后，收益按贡献权重按比例分配，全程链上可审计。
>
> 这就是全 Agent 公司的样子。

**转场**：淡出 → Logo 淡入。

---

## 第 6 段：收尾（3:10–3:30）

**画面**：Trading Guild Logo 居中，三个 Agent 头像环绕。文字叠层：「Casper AI Toolkit」「x402 · MCP · Odra · CSPR.cloud」「GitHub 开源」。淡出。

**解说**：

> Casper 自主交易公会——AI Agent 不辅助人类，而是自主交易、协作、创造价值。
>
> 基于 Casper AI Toolkit：x402 做 Agent 支付，MCP 做工具集成，Odra 做合约，CSPR.cloud 做数据。
>
> 代码开源，架构可扩展。感谢观看。
